# The Syrian Mosaic Foundation Website

A minimalist, journalism-inspired website showcasing the Syrian Mosaic Foundation's mission to preserve Syria's cultural heritage and build bridges across communities.

## Design Philosophy

- **White & Spacious**: Clean, minimalist design with generous whitespace
- **Typography-First**: Clear hierarchy using Playfair Display (headings) and Inter (body)
- **Accessibility**: WCAG AA compliant with proper semantic HTML and keyboard navigation
- **Responsive**: Mobile-first approach ensuring great experience on all devices
- **Performance**: Minimal JavaScript, optimized assets, fast loading times

## Pages

- **Home** (`index.html`) - Mission, vision, and strategic initiatives
- **Impact** (`impact.html`) - Key outcomes, metrics, and initiative highlights
- **Tours** (`tours.html`) - Cultural tour offerings and booking information
- **Team** (`team.html`) - Founders and team members
- **Media** (`media.html`) - Press coverage and media kit
- **Contact** (`contact.html`) - Contact form and information

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom properties (CSS variables), Flexbox, Grid
- **JavaScript**: Minimal, vanilla JS for simple interactions
- **Fonts**: Google Fonts (Playfair Display, Inter)

## Color Palette

- Navy: `#1A3A52` - Primary brand color
- Gold: `#D4A574` - Accent color
- Terracotta: `#C77E5C` - Secondary accent
- White: `#FFFFFF` - Primary background
- Cream: `#FAFAF8` - Alternate background
- Text: `#2C2C2C` - Body text
- Muted: `#6B7280` - Secondary text

## Local Development

Simply open any HTML file in a modern web browser. No build process required.

For local server:
```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Content Updates

Content is managed directly in HTML files. Key sections:

- Mission/Vision: `index.html` 
- Team roster: `team.html`
- Media coverage: `media.html`
- Tour packages: `tours.html`
- Statistics: `impact.html`
