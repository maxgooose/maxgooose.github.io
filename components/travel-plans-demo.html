How to Use This Component

1. Copy the HTML

Copy the section between the "COPY FROM HERE" comments into your page.

2. Customize the Cards

For each .travel-card, update:

  • --card-bg - Background image URL
  • href - Link destination
  • Duration text (e.g., "10 Days")
  • Price text (e.g., "$3,500")
  • Title text

3. Hover Effects

The cards have these interactive behaviors:

  • Lift: Card moves up 10px on hover
  • Expand: On desktop (55em+), hovered card grows to 190% width
  • Fade: Non-hovered cards' content fades out
  • Smooth timing: 0.875s cubic-bezier easing

4. CSS Location

The styles are in styles.css under the "Travel Plans Section" heading.
